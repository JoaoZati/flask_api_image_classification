Objetivo da aplicação

O objetivo dessa aplicação é aplicar o conhecimento de geração de api com flask, utilização do banco de dados MongoDB, conhecimentos básicos da biblioteca tensorflow e o deploy da aplicação utilizando docker na AWS.


Se gostou da aplicação tem uma similar no meu github que seria uma simulação de uma conta bancária github.com/JoaoZati/flask_api_bank_account


Se gostou da minha aplicação e deseja mais tokens entre em contato joaozati@gmail.com


Nota: se você não sabe lidar com apis, basta baixar o aplicativo Postman e enviar um POST com um body contendo um json com as informações abaixo. Para mais detalhes entre em contato.


Comandos


/register -> registra um usuario para utilizar, no corpo deve conter: username, password


/classify -> Retorna o classificação de uma imagem que você mandar, consome 1 token e no corpo deve conter: username, password e url (url deve ter no final .jpg)


/Refil -> utilizado pelo admin para adicionar tokens (qualquer duvida entres em contato no email joaozati@gmail.com)
